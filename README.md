XAI Medical Insurance Cost Prediction

This repository contains the code, dataset and README for the final assignment for course xAI.

The Python file expects a the insurance database, it is included in the repository. It should be placed in the same directory as the Python file.

Installation:
Install all required packages listed below:
  1. torch
  2. pandas
  3. numpy
  4. captum
  5. scikit-learn
  6. lime

Usage guide:
The script does all the dataprocessing, training of the model, and evaluation automatically. All you need to do is run the Python file in any IDE and the script will return all final results that are also included in the report.
